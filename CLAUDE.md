# CLAUDE.md

Helios Fusion — multi-signal algorithmic trading bot for Alpaca (Paper).
Runs on Hetzner VPS, controlled via Telegram, started by cron on trading days.

## Architecture

```
Phone (Telegram) ←→ telegram-ctl.js (systemd)
                       ↓ PM2 commands
pre-market-scan.js (9:50 ET) → market-context.json
                       ↓
helios-bot.js (1005–1110 ET)
                       ↓ REST API
         Alpaca API ←→ Market data + order execution
```

## Commands

```bash
npm start          # node scripts/helios-bot.js
npm run pre-market # Pre-market scanner
npm run backtest   # Helios Fusion backtest
npm test           # Run all test suites

node scripts/helios-analytics.js  # Performance report

```
